# LitChat

## 简介
使用 Express + socket.io + MongoDB 搭建的轻量化聊天室

## 开始
```
git clone https://github.com/Eric-Cool/LitChat.git
npm install
```

## 运行
```
cd LitChat
node .\bin\www
```

## 功能
- [x] 登录注册
- [x] 在线聊天
- [x] 获取在线用户列表
- [x] 进入退出提醒
- [x] 聊天时间轴
- [x] 添加表情
- [x] 发送图片
- [x] 用户私信
- [x] 获取用户IP
- [x] 用户自定义设置
- [x] 管理员权限：封禁/解封
- [x] 显示历史聊天记录
- [x] 公告功能
